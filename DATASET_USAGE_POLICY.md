# BFSET Dataset Usage Policy  

The BFSET dataset is the exclusive property of Cognisoft.

## 1. Access Restrictions
The dataset is not publicly available. Access may be granted only:
- After acceptance of the BFSET publication, or
- Under a signed Non-Disclosure Agreement (NDA),
- For academic research purposes only.

## 2. Prohibited Actions
- Public redistribution of the dataset
- Sharing images or annotations
- Hosting the dataset on external platforms
- Commercial use
- Removing or altering dataset metadata
- Using the dataset name “BFSET” without authorization

## 3. Allowed Actions (after authorization)
- Internal academic research
- Reproduction of experiments described in the article
- Training models using the dataset within an authorized environment

## 4. Citation Requirement
Any work using the dataset must cite:

“Hamda Elbahri, BFSET: A Beard-Focused Dataset for Soft Biometrics, Cognisoft, 2025.”
